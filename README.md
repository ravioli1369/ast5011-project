## Installation

### Prerequisites

Install [uv](https://docs.astral.sh/uv/getting-started/installation/).

### Setup

```bash
git clone git@github.com:ravioli1369/ast5011-project.git
cd ast5011-project
uv sync
```

### Usage

Activate the environment:

```bash
source .venv/bin/activate
```

Run scripts as usual through python:
```bash
python path/to/script.py
```

Run scripts through `uv run` if you don't have the environment activated:

```bash
uv run path/to/script.py
```

> **Note:** Use the `ast5011-project (3.12.12) (Python 3.12.12)` kernel while running Jupyter notebooks.
